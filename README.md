
# EXP_Tracker_App
Cross-platform Expense Tracker built with Flutter. Enables users to add, delete, and manage expenses with dynamic chart visualization. Features responsive design, Light/Dark themes, and support for Android, iOS, macOS, and Web. Built using Dart, Material 3, and Stateful Widgets.
